# DayDayNews
仿网易新闻客户端，实现新闻浏览，视频播放，抓取百度图片，瀑布流显示

##Update Log

- 2016-1-2 修改了首页顶部滚动条的接口 <br />
- 2016-1-14 处理天气预报加载时间长，没页面显示的问题。<br />
- 2016-1-19 更换了corelocation定位，系统定位繁琐速度慢。更换为INTULocationManager第三方定位，block调用简单有效<br />
